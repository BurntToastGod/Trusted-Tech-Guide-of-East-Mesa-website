# SOP — Appointment Flow (Internal)

## Before the visit
- Confirm time/location and top goal.
- Send `Customer_Prep_Checklist.md` if needed.
- Prepare: charger bag, notepad, spare cables.

## During the visit
- Start with “What would make today a win?”
- Fix the urgent issue first.
- Teach + practice: have the client repeat the steps.
- Write simple notes they can keep.
- Do a short safety sweep if relevant.

## After the visit
- Create the after-visit summary.
- If follow-up is needed, schedule a next step.
- Ask for a review (once you have a Google profile link).
