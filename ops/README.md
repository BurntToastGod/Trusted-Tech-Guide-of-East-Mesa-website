# Operations Files (Jobs + Customer Packets)

This folder is your working “job system” for running real appointments.

## How to use (simple flow)
1. Copy `ops/templates/Job_Packet_Template.md` into `ops/jobs/YYYY-MM/Client_Lastname_Firstname/`
2. Fill it out during the call/text and the visit.
3. After the visit, create a customer-friendly handout from:
   - `ops/templates/Customer_After_Visit_Summary.md`
   - `ops/templates/Customer_How_To_Notes.md`
4. Store only what you actually need. Avoid sensitive data whenever possible.

## Quick safety note
Do not store raw passwords in these files. Use the password placeholders provided.
