# Trusted Tech Of East Mesa — Simple Website

This folder contains a single-page static website you can host anywhere.

## Quick preview (no installs)
Open `index.html` in a browser.

## Host it (easy options)
- Drag-and-drop host: Netlify / Cloudflare Pages / GitHub Pages
- Any web host: upload `index.html` + `styles.css`

## Customize (common edits)
- Phone number and tagline are near the top of `index.html`
- Colors and fonts are in `styles.css` under `:root`
